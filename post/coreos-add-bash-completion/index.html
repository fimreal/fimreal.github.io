CoreOS 添加 bash-completion 命令补全

2019-04-16
Linux
约 251 字 预计阅读 1 分钟

文章目录

CoreOS 系统常用作容器宿主机系统，开发者本着缩小镜像体积提供核心功能的初衷，没有附带上命令补全功能，这可真要逼死 TAB 键按个不停随时感觉要飞起来的 bash shell 用户。 本文是参考 github/coreos 上 ISSUE 提供的办法来实践测试写出，最终解决方案有些许优化。

方法说明

这里要利用 toolbox 小工具安装好 bash-completion 环境，然后把生成的脚本导入 coreos 中。 PS: toolbox 是 Coreos 提供的一个用来扩展系统的沙盒环境，默认使用 Fedora 系统。使用 Ctrl+D 或者按 Ctrl+] 三次可以退出。安装的包都会保存下载供长期使用。

解决过程

在 shell 中依次执行：

$ toolbox
# dnf -y install bash-completion wget
# curl  https://raw.githubusercontent.com/docker/cli/master/contrib/completion/bash/docker -o /usr/share/bash-completion/completions/docker
# cp -R /usr/share/bash-completion /media/root/var/
# exit
$ mkdir /etc/bash/bashrc.d/
$ echo "source /var/bash-completion/bash_completion" >> /etc/bash/bashrc.d/bash-completion.sh
